# Input Artifacts

## Required

1. Prospect CSV
2. Offer definition from the user
3. Goal for the campaign

## Strongly Recommended

1. Proof-of-work doc or case-study doc
2. Existing copy or tone example
3. Output folder for final files

## Useful CSV Fields

Best fields for this workflow:
- first name
- full name
- last name
- job title
- LinkedIn URL
- person headline
- company name
- company domain
- company website
- industry
- company description
- employee count
- technologies
- revenue range
- existing email
- email status

## User-Provided Files

Reference files may be:
- already provided in the current task
- provided later after inspection

Typical examples:
- raw lead list CSV
- proof-of-work `.docx`
- offer notes
- existing cadence `.docx`
- export from Clay/Prospeo/Apollo

If a proof file is mentioned but not yet supplied, ask for it before writing final copy.
