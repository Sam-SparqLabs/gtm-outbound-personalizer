# Proof Extraction

## Goal

Select proof that is:
- specific
- credible
- close to the current offer
- easy to drop into cold email

## Prefer In This Order

1. Meeting-booked outcomes
2. Pipeline or ARR growth outcomes tied to outbound
3. Named customer results
4. Cost/ramp replacement proof
5. Risk reversal if available

## Good Proof Examples

- `102 qualified meetings booked in 1 month for a funded SaaS client`
- `10-20 meetings per day from the same outbound model`
- `6x growth from $5M to $30M ARR in 36 months`

## Avoid Leading With

- vague agency claims
- broad brand lists without outcome
- proof that does not match the audience

## Output Columns To Support Copy

At minimum create:
- `Best Proof Point`

If the list is segmented by audience, optionally create:
- `Proof Line`
- `Risk Reversal Line`

## Source Handling

Treat user-provided proof docs as the source of truth for claims.
If the user asks for “juicy” proof, choose the strongest specific proof that can survive scrutiny.
