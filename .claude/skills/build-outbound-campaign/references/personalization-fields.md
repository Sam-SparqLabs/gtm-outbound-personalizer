# Personalization Fields

## Required Fields To Add

- `Primary Email`
- `Primary Email Source`
- `Primary Email Verified`
- `Email QC Status`
- `Clean First Name`
- `Clean Last Name`
- `Clean Full Name`
- `Clean Job Title`
- `Clean Company Name`
- `Display Company Name`
- `Role Bucket`
- `Personalization Angle`
- `Pain Hook`
- `Growth Hook`
- `Best Proof Point`
- `Tailored Message`

## Useful Additional Fields

- `Short Intro`
- `First Line`
- `Proof Line`
- `Risk Reversal Line`
- `Send Priority`

## Role Bucket Guidance

Common buckets:
- `founder_exec`
- `revenue_gtm`
- `tech_ops`
- `ops_delivery`
- `hr_talent`
- `general_exec`

## Angle Guidance

Common angles:
- predictable pipeline without SDR ramp
- delivery busy, pipeline thin
- ramp without waiting for hires
- calendar full of qualified meetings
- pipeline not empty

## Message Style

Personalization should sound researched but scalable.
Avoid:
- fake precision
- unsupported assumptions
- long first lines that read like a memo
